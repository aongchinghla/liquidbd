# LIQUID TSHIRT

A Bangladesh-focused premium t-shirt ecommerce UI built with Next.js, TypeScript, Tailwind CSS, GSAP, and Lucide.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Included
- Modular component structure
- Dark premium homepage
- LocalStorage cart
- Login/signup UI
- Checkout modal
- Bangladesh-focused messaging
